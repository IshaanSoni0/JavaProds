3.1  Probability of Double-Crit Substats
─────────────────────────────────────────────────────────────────────────────

An artifact's substats are drawn one at a time from the weighted pool,
without replacement — once a substat is drawn it cannot appear again on
that piece. To find the probability that both CR% and CD% appear among
the four draws, apply the sequential probability rule.

Take a Flower as a worked example. Flat HP is always the main stat, so it
is excluded from the substat pool. The remaining nine substats have a
total pool weight of W = 38, with w_CR = w_CD = 3.

The probability that CR% is drawn first and CD% is drawn second is:

  P(CR% 1st, CD% 2nd) = (w_CR / W) × (w_CD / (W − w_CR))
                      = (3/38) × (3/35) = 9/1330 ≈ 0.68%

Since w_CR = w_CD, the reverse order (CD% first, CR% second) gives the
same value. Both orderings combined give ~1.35%.

The two crit substats can also occupy any of the other C(4,2) − 1 = 5
remaining position pairs in the four-draw sequence (positions 1&3, 1&4,
2&3, 2&4, and 3&4). For each later pairing, a non-crit substat has been
drawn at an intermediate position, which changes the pool weight before
the second crit sub is drawn. Summing all six position-pair contributions
with their respective sequential probabilities gives the full analytical
result. For a Flower this equals approximately 9.2%, confirmed by
simulation to 9.15%.

Table 1 shows the results for each slot. The pool weight W varies because
the main stat is excluded from the substat pool. Note that Hydro DMG
Bonus% is only a main stat, not a substat, so a Goblet with Hydro DMG%
main does not remove anything from the substat pool (W = 44).

  Table 1: Double-crit probability by slot
  ┌──────────────────────────────┬────┬──────────────────────┐
  │ Slot / main stat             │  W │  P(double-crit)      │
  ├──────────────────────────────┼────┼──────────────────────┤
  │ Flower (flat HP)             │ 38 │   9.15%  (sim.)      │
  │ Plume (flat ATK)             │ 38 │   9.18%  (sim.)      │
  │ Sands (HP% main)             │ 40 │   8.34%  (sim.)      │
  │ Goblet (Hydro DMG% main)     │ 44 │   7.69%  (sim.)      │
  │ Circlet (non-crit main)      │ 40 │   6.40%  (sim.)      │
  │ Circlet (CD% main)           │ 41 │   0%     (exact)     │
  └──────────────────────────────┴────┴──────────────────────┘

For Neuvillette's Circlet, CD% is the main stat. Because the main stat is
excluded from the substat pool, CD% cannot appear as a substat on the
same piece. P(double-crit | CD% main) = 0% exactly, for any number of
pieces farmed.

3.2  Upgrade Roll Distribution
─────────────────────────────────────────────────────────────────────────────

Once the four substats are set, the piece gains five upgrade events at
+4, +8, +12, +16, and +20. At each event, one of the four substats is
selected uniformly at random (25% each) and receives one additional roll.

The number of upgrades a specific substat receives over five events
follows a Binomial distribution:

  X ~ Bin(5, 0.25)

Table 2 shows the full distribution (Genshin Impact Wiki,
Artifact/Distribution):

  Table 2: Upgrade count distribution for one substat  [X ~ Bin(5, 0.25)]
  ┌───────────────────┬─────────────┐
  │ Upgrades received │ Probability │
  ├───────────────────┼─────────────┤
  │        0          │   23.73%    │
  │        1          │   39.55%    │
  │        2          │   26.37%    │
  │        3          │    8.79%    │
  │        4          │    1.46%    │
  │        5          │    0.10%    │
  └───────────────────┴─────────────┘

Getting 3 or more upgrades into a single substat — the typical standard
for a strong substat — happens only about 10.4% of the time. For a
double-crit piece, each crit substat still sees this distribution
independently and competes against two non-crit substats for every
upgrade. The probability that both CR% and CD% each receive at least
2 upgrades is:

  P(CR% ≥ 2 upgrades) × P(CD% ≥ 2 upgrades) = 0.357 × 0.357 ≈ 12.7%

Combined with the ~9.2% chance of double-crit in the first place (for a
Flower), this means a Flower with at least 2 upgrades in both crit
substats occurs in about 9.2% × 12.7% ≈ 1.2% of pieces. This confirms
that high CV on a single piece is genuinely rare.

─────────────────────────────────────────────────────────────────────────────
3.3  Simulation and CV Distribution
─────────────────────────────────────────────────────────────────────────────

A Java simulation (ArtifactSim.java) generated N = 5,000,000 artifacts
per slot, implementing the game mechanics above: weighted draws without
replacement, the 80%/20% three-stat/four-stat starting split, and the
Binomial upgrade distribution. Only CR% and CD% substat values are
recorded per piece; all other substats are discarded.

Crit Value (CV) metric

To combine CR% and CD% into a single score, define:

  CV = k × CR% + CD%

where k weights CR% relative to CD%. From Section 2, the average substat
roll values are avrCR = 3.305% and avrCD = 6.608%, giving:

  k = avrCD / avrCR = 6.608 / 3.305 = 2.000

So CV = 2 × CR% + CD%. This metric counts crit budget in CD%-equivalent
units: one expected CR roll is worth exactly half of one expected CD roll,
and the constraint CV = 2 × CR + CD is a statement of roll investment —
any two builds with the same CV represent the same total number of
expected crit rolls.

Key findings  (N = 5,000,000 per slot)

P(CV = 0) — no crit substats at all:

  Flower / Plume:      ~40.9%
  Sands / Goblet:      ~47.9%
  Circlet (CD% main):  ~68%

On average, only about 1 in 2 pieces has any crit value. For a CD% main
Circlet, 2 in 3 pieces have no crit substats at all, since the only
available crit substat is CR%.

Mean CV (substat CV only, all pieces including CV = 0):

  Flower / Plume: ~9.26     Sands: ~8.89
  Goblet: ~8.53             Circlet (CD% main): ~7.89

These low means are largely driven by the ~40–68% of pieces with CV = 0.
Among pieces with at least one crit substat, mean CV rises to ~15.2–15.6
for Flower, Plume, Sands, and Goblet.

Histogram and KDE distribution charts for each slot are included in
Appendix B.

3.4  Realistic Crit Constraint from Farming
─────────────────────────────────────────────────────────────────────────────

Rather than setting an arbitrary CV threshold for "a good piece", this
section derives the expected best CV achievable with a fixed farming
investment. The expected-maximum function E[max of N independent samples]
reads directly off a simulation curve: given N attempts, it gives the
expected highest CV found.

Farming benchmark

A typical two-week farming period yields:

  14 days × 200 resin/day = 2,800 resin
  ÷ 20 resin per 5-star artifact      ≈ 140 pieces total
  ÷ 5 slot types                      ≈ 28 pieces per slot
  → rounded to N = 30 per slot

Total substat CV budget across all five pieces:

  32.5 + 32.5 + 31.2 + 31.0 + 23.4 = 150.6

  Total_CV = 150.6 + 62.2 = 212.8 ≈ 213

This value of Total_CV = 213, combined with k = 2 derived in Section 3.3,
is the constraint fed into the optimisation in Section 4.

   4.4  Crit rate vs crit damage optimisation
        - Constraint from Section 3: k × CR + CD = Total_CV

          the community. The derivation IS the IA contribution here.

Genshin Impact is a third person action game where players build teams of
characters progress through the games story and clear combat challenges.
Over my three years of playing, I have spent a lot of time trying to make
my characters deal as much damage as possible to clear the hardest game
modes: Spiral Abyss, Imaginarium Theater, and Stygian Onslaught. This
process, typicaly called build optimisation, involves choosing the right
equipment for each character.

My main damage dealer is Neuvillette. What makes him interesting to
analyse is that his damage depends on several mechanics at once: a passive
ability that changes his damage bonus as his HP drops during an attack, an
artifact set that changes his critical hit rate as his HP fluctuates, and
a damage formula that scales directly with his maximum HP. Together, these
create a system where these mechanics interact, and all affect the final
damage.

Aim: The aim of this exploration has two parts: to model the probability
distribution of artifact substat rolls and determine what CRIT stats are
realistically achievable, and to use a mathematical damage model to find
the optimal CRIT Rate to CRIT DMG ratio and goblet main stat for
Neuvillette.

================================================================================
  SECTION 2 — GAME MECHANICS OVERVIEW  [DRAFT]
================================================================================

Every hit in Genshin Impact follows the same general formula
(Genshin Impact Wiki, Damage):

  DMG = Base DMG × (1 + DMG Bonus) × DEF Multiplier
                 × RES Multiplier × CRIT Multiplier

Base DMG is the raw damage before any modifiers. For Neuvillette, his main
attack scales with his maximum HP, so:

  Base DMG = M × HP_max     (M = 12.51% per tick at talent level 8)

where M is the talent multiplier and HP_max is his total maximum HP.

DMG Bonus is the sum of all percentage damage bonuses from equipment and
passive abilities, expressed as a decimal (for example, +30% DMG Bonus
adds 0.30). These bonuses add together before being multiplied into the
formula.

DEF Multiplier accounts for the enemy's defence. The formula is
(Genshin Impact Wiki, Damage):

             Lc + 100
  DEF Mult = ─────────────────────────────
             (Le + 100) + (Lc + 100)

where Lc is the character's level and Le is the enemy's level.
For Neuvillette at level 80 fighting a level 100 enemy (standard for
endgame content like Spiral Abyss):

         180        180
  DEF = ─────── = ─────── ≈ 0.474
        200+180     380

RES Multiplier accounts for the enemy's elemental resistance. For a
typical enemy with 10% Hydro RES:

  RES Mult = 1 − 0.10 = 0.90

CRIT Multiplier is where CRIT Rate (CR) and CRIT DMG (CD) come in.
CRIT Rate is the probability that a hit deals a critical strike. CRIT DMG
is the percentage bonus to damage when a critical strike occurs. A
non-critical hit has a multiplier of 1; a critical hit has a multiplier
of (1 + CD). Since each hit is an independent event with success
probability CR, the expected value of the CRIT Multiplier is:

  E[CRIT Mult] = (1 − CR) × 1 + CR × (1 + CD) = 1 + CR × CD

Putting it all together, the expected damage per tick is:

  E[D_tick] = K × (1 + CR × CD)

  where  K = M × HP_max × (1 + DMG Bonus) × DEF Mult × RES Mult

K consists of every term that is fixed for a given build: the talent
scaling, the max HP, all additive damage bonuses, and both defensive
multipliers. Since these variables are not easly changeable, K is a
constant, and Optimising the artifact substats therefore reduces entirely
to maximising the crit multiplier.

The goblet comparison (Section 4.5) changes which goblet main stat is
used, which changes K since different goblets change either max HP or DMG
Bonus. But the CRIT optimisation that uses substats is unchanged
regardless of which goblet is equipped since the optimal CR and CD ratio
depends only on the substat budget, not on K. The two questions can
therefore be answered independently and combined at the end.

Neuvilette's passive talent, "Discipline of the Supreme Arbitration",
adds a Hydro DMG Bonus that depends on his current HP percentage. The
exact in-game text reads:

    "For each 1% of Neuvillette's current HP greater than 30% of Max HP,
     he will gain 0.6% Hydro DMG Bonus. A maximum bonus of 30% can be
     obtained this way."
    (Genshin Impact, in-game passive description)

Expressing this as a function of h, where h is the current percent of HP:

  PassiveBonus(h) = 0.006 × (100h − 30)   for 0.30 < h < 0.80
  PassiveBonus(h) = 0                      for h ≤ 0.30

The cap of 30% is reached at h = 0.80 (exactly 80% HP):

  0.006 × (80 − 30) = 0.30  ✓

During a full Charged Attack, HP drains from 100% down to approximately
50%. This creates two distinct phases for the passive:

  Phase 1 (100% → 80% HP):  Bonus is at its maximum of 30% throughout.
  Phase 2  (80% → 50% HP):  Bonus falls continuously from 30% to 12%.

Phase 2 will require integration over the attack duration to find the
average damage bonus.

Artifacts are the main equipment system in Genshin Impact. There are five
slots: Flower (always gives flat HP as its main stat), Plume (always gives
flat ATK), Sands, Goblet, and Circlet. Each piece also has up to four
substats chosen from a shared pool.

The Sands, Goblet, and Circlet have randomly assigned main stats, and
substats drawn from a weighted pool specific to each slot. Substats are
drawn sequentially from a pool of ten possible stats, excluding whichever
stat is already the main stat. Each substat has a weight that controls
the probability of being chosen (Genshin Impact Wiki, Artifact/Distribution):

  ┌──────────────────────┬────────┐
  │ Substat              │ Weight │
  ├──────────────────────┼────────┤
  │ Flat HP              │   6    │
  │ Flat ATK             │   6    │
  │ Flat DEF             │   6    │
  │ HP%                  │   4    │
  │ ATK%                 │   4    │
  │ DEF%                 │   4    │
  │ Energy Recharge%     │   4    │
  │ Elemental Mastery    │   4    │
  │ CRIT Rate%           │   3    │
  │ CRIT DMG%            │   3    │
  └──────────────────────┴────────┘

CRIT Rate% and CRIT DMG% each have the lowest weight in the pool. On a
Flower, for example, the total pool weight is 3+3+6+4+4+4+4+4+6+6 = 44
(flat HP excluded as main stat). The chance of any one draw being a
specific crit stat is only 3/44 ≈ 6.8%. Because only 4 substats are
chosen, most artifact pieces will not have both CRIT Rate and CRIT DMG
as substats.

The four substat draws happen without replacement. Once a substat is
chosen, it is removed from the pool for that piece. Each time a substat
is rolled (either at the start or during an upgrade), it lands on one of
four value tiers with equal probability (25% each). For CRIT Rate% and
CRIT DMG%, the tiers are (Genshin Impact Wiki, Artifact/Stats):

  ┌────────┬────────────┬────────────┐
  │  Tier  │ CRIT Rate% │ CRIT DMG%  │
  ├────────┼────────────┼────────────┤
  │   1    │   3.89%    │   7.77%    │
  │   2    │   3.50%    │   7.00%    │
  │   3    │   3.11%    │   6.22%    │
  │   4    │   2.72%    │   5.44%    │
  └────────┴────────────┴────────────┘

Every CRIT DMG% tier value is exactly twice the corresponding CRIT Rate%
tier value. The average roll for each stat is:

  avrCR = (3.89 + 3.50 + 3.11 + 2.72) / 4 = 13.22 / 4 = 3.305%
  avrCD = (7.77 + 7.00 + 6.22 + 5.44) / 4 = 26.43 / 4 = 6.608%

So avrCD = 2 × avrCR exactly. This ratio is how the CRIT Value metric
is defined.

An artifact starts with either 3 substats (80% of pieces) or 4 substats
(20%) and is upgraded 5 events total. At each event, one existing substat
is selected at random (each substat has a 25% chance) and receives one
additional roll at a random tier. For pieces that start with 3 substats,
the first event instead adds the fourth substat from the remaining
weighted pool.

Because each upgrade picks uniformly from 4 substats, the number of times
any given substat gets upgraded follows a Binomial distribution:
BinomPDF(5, 0.25, X). This makes hitting a high CRIT value on any single
piece unlikly, which is exactly what the simulation in Section 3 shows.

Marechaussee Hunter is the recommended artifact set for Neuvillette. Its
bonuses are (Genshin Impact, in-game set description):

  2-piece: Normal and Charged Attack DMG +15%
  4-piece: "When current HP increases or decreases, CRIT Rate will be
            increased by 12% for 5s. Max 3 stacks."

The 4-piece bonus interacts with Neuvillette's HP drain mechanic. Each
hit of his Charged Attack reduces his current HP, which counts as an HP
change event. This means all three stacks are made and maintained
throughout the attack, adding +36% CRIT Rate to his effective total, and
This +0.36 contribution from the set is accounted for in the optimisation
in Section 4.
